## ü§ñ About AIResponder

**AIResponder** is an intelligent AI auto-responder plugin for Vencord that uses OpenRouter.ai with advanced language models including Llama, Qwen, and other state-of-the-art AI models.

### Key Features
- ü§ñ Intelligent AI responses to Discord DMs
- üåç Global DM mode for automatic responses
- üîë Custom API key support
- üí¨ Conversation memory and context
- ‚öôÔ∏è Customizable settings and preferences
- üö´ User blacklist functionality
- üìä Usage monitoring and rate limit handling

### Technology Stack
- **Frontend**: TypeScript, React
- **Framework**: Vencord Plugin System
- **AI Provider**: OpenRouter.ai
- **Models**: Llama 3.1, Qwen 2.5, and others
- **Platform**: Discord
